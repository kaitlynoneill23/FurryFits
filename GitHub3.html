  5. Check the box for "Initialize this repository with a README" (optional).
  6. Click "Create repository".

Step 2: Upload Your Website Files

Option 1: Using GitHub Website

  1. In your repository, click "Add file" > "Upload files".
  2. Drag and drop your HTML, CSS, JavaScript files.
  3. Click "Commit changes".

Option 2: Using Git & Command Line (Recommended)

  1. Install Git if you haven't.

  2. Open a terminal and navigate to your project folder.

  3. Run these commands:

    sh
    git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/YOUR-REPOSITORY.git
git push -u origin main

Step 3: Enable GitHub Pages

  1. Go to your repository on GitHub.
  2. Click "Settings" (⚙️) at the top.
  3. In the left menu, click "Pages".
  4. Under "Branch", select main (or master).
  5. Click "Save".

Step 4: Get Your Live Website Link

  1. Wait a few minutes for GitHub to deploy your site.

  2. Refresh the GitHub Pages section.

  3. Your live link will be in the format:

    arduino
    https://your-username.github.io/your-repository/

Optional: Custom Domain

If you want a custom domain, you can:

  1. Buy a domain (e.g., from Namecheap, GoDaddy).
  2. Add a CNAME file in your repo with your domain name.
